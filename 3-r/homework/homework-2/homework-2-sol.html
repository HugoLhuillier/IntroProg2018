Exercice 2: scaling to \((0,1)\)

Write a function rescale that takes a vector as input and returns a corresponding vector of values scaled to lie in the range 0 to 1.

scale_unit <- function(x) {
  # rescales x -- a vector -- to lie in the range 0 to 1 
  # identifies the lower and upper bound of the vector
  lower_bound <- min(x)
  upper_bound <- max(x) 
  # rescale the vector to lie in (0,1)
  scaled_x    <- (x - lower_bound) / (upper_bound - lower_bound)
  return(scaled_x) 
}

Exercice 4: scaling to \((a,b)\)

Write a function scale that takes a vector with a range \((0,1)\) as input and returns a vector of values scaled to lie in the range a to b.

Note: if the original vector does not lie in the \((0,1)\) interval, warn the user that the computation is erroned.

scale <- function(x, a, b) {
  # scales x -- a vector that lies in the (0,1) interval -- to lie in the range a to b
  if (min(x) != 0.0 | max(x) != 1.0) {
    warning("function designed for vectors lying in the unit interval")
  }
  scaled_x    <- x * (b - a) + a
  return(scaled_x) 
}

Exercice 5: composition

Write a function that scales an arbitrary vector x to a range a and b, where a and b are arguments of the function with default value 0 and 1.

rescale <- function(x, a, b) {
  # rescale an arbitrary vector - x - to an interval (a,b)
  # first, rescale x to lie on the unit interval
  unit        <- scale_unit(x)
  # then, scale unit to the (a,b) range
  scaled_x    <- scale(unit,a,b)
  return(scaled_x) 
}
